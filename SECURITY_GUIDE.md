# 安全功能使用指南
## 📋 概述
装箱记录管理系统现已添加完整的安全认证功能，支持管理员和访客两种访问模式，确保数据安全与访问便利的平衡。

## 🔐 用户角色

### 管理员 (admin)
- **权限**: 完整的增删改查权限
- **功能**: 可以添加记录、删除记录、修改佣金设置、查看所有数据
- **认证**: 需要登录验证

### 访客 (guest)  
- **权限**: 仅查询权限
- **功能**: 只能查看统计数据和记录列表，无法进行任何修改操作
- **认证**: 无需登录，默认访问模式

## 🛠️ 初始设置

### 1. 创建管理员账户

首次使用系统时，管理员账户会自动创建：

1. **访问登录页面**: 打开系统后在右上角点击"管理员登录"进入登录页面
2. **自动检测**: 系统检测到没有管理员账户时，会自动弹出创建窗口
3. **填写信息**: 输入管理员用户名、密码（至少6位）和确认密码
4. **自动完成**: 系统会自动创建管理员账户并登录到系统

⚠️ **重要**: 请妥善保管管理员账户信息，这是系统安全和数据保护的钥匙。

### 2. 数据库初始化

用户表将随系统自动创建，管理员账户信息将安全存储在SQLite数据库中。

## 🖥️ 使用方法

### 访客访问

1. **访问系统**: 直接在浏览器中打开应用
2. **默认模式**: 系统自动以访客身份访问
3. **权限限制**: 
   - 隐藏"添加记录"表单
   - 隐藏所有"删除"按钮
   - 隐藏"佣金设置"按钮
   - 统计信息只读显示

4. **切换身份**: 点击右上角"管理员登录"可切换到管理员身份

### 管理员访问

#### 登录方式一：通过登录页面
1. 访问登录页面 `/login`
2. 输入用户名和密码
3. 点击"登录"按钮

#### 登录方式二：通过系统内弹窗
1. 以访客身份进入系统
2. 点击右上角"管理员登录"按钮
3. 在弹窗中输入用户名和密码

#### 管理员权限
- 🔓 显示"添加记录"表单
- 🔓 显示"删除"按钮
- 🔓 显示"佣金设置"按钮
- ✅ 完整的添加、删除、修改权限

### 安全退出

管理员完成操作后，建议及时退出登录：
1. 点击右上角"退出登录"按钮
2. 确认退出操作
3. 系统将清除会话信息
4. 自动切换回访客模式

## 🔒 安全特性

### 密码安全
- 使用安全的密码哈希算法（Werkzeug）
- 密码传输过程中加密处理
- 数据库中只存储密码哈希值

### 会话安全  
- 使用安全的会话密钥
- 会话有效期1小时
- 登出后自动清理会话信息

### 权限控制
- RESTful API级别的权限验证
- 前端界面级别的权限控制
- 双重保护确保数据安全

## 🚨 安全建议

1. **强密码设置**
   - 使用包含字母、数字、特殊字符的复杂密码
   - 定期更换管理员密码
   - 不要在多个系统使用相同密码

2. **安全操作**
   - 离开时及时退出登录
   - 不要在公共电脑上保存登录状态
   - 发现异常访问立即修改密码

3. **数据备份**
   - 定期备份数据库文件（app.db）
   - 妥善保管管理员账户信息
   - 建议设置密码提示但不记录明文密码

## 🔧 故障排查

### 忘记管理员密码
目前系统不支持密码找回功能，如果忘记密码需要：
1. 备份现有数据库
2. 使用数据库工具手动重置密码
3. 或重新创建管理员账户（会丢失现有用户信息）

### 登录失败
- 检查网络连接
- 确认用户名和密码正确
- 检查浏览器Cookie设置
- 重启应用尝试

### 权限异常
- 刷新页面重新获取权限
- 退出登录后重新登录
- 清除浏览器缓存
